# day-2
This Python program validates StudentId, Email Id, password and referral code.
If all inputs match the required conditions , it prints APPORVED or otherwise REJECTED.

Validation Rules:
 Student ID : CSE- and 3 digits.
 Email Id: contains "@" and "." ends with .edu
 Password: min 8 character ,starts With uppercase ,has at least one digit.
 Referral code:Starts with REF , 2 digits ,ends with "@".
Outputs:
 APPORVED -> all validations pass
 REJECTED -> any validation fails
Concepts used:
 Strings,conditional statements,logical operator,input handling.
